# Task 1: Linear Regression using California Housing Dataset

## Overview

This project demonstrates the implementation of a Linear Regression model using the California Housing Dataset from Scikit-learn.

## Objectives

- Load and explore the dataset
- Perform Exploratory Data Analysis (EDA)
- Train a Linear Regression model
- Evaluate the model using MAE, RMSE, and R² Score
- Visualize the results

## Dataset

- California Housing Dataset
- Source: Scikit-learn
- Records: 20,640
- Features: 8
- Target: House Price

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Files

- `task1_ml_linear_regression.ipynb` – Complete implementation
- `Report.pdf` – Project summary
